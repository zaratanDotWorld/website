That pattern repeats at the page and query level below, not just in the aggregate.

## What is working

- **"settle living" query** — 6 clicks / 358 impr / 1.68% CTR / pos 5.5, up from 5 clicks / 308 impr / 1.62% / pos 6.2. Position and volume both improved.
- **"settle living reviews"** — 3 clicks / 39 impr / 7.69% CTR / pos 3.9, up from 1 click / 34 impr / 2.94% / pos 4.4. Strong CTR at a strong position.
- **`/about`** — 12 clicks / 123 impr / 9.76% CTR / pos 5.0, up from 4 clicks / 95 impr / 4.21% / pos 4.8. Clicks tripled, CTR more than doubled, on a flat position.
- **www/apex canonicalization looks resolved.** The prior report's #1 finding was `/houses/sage` and `/about` splitting traffic across `zaratan.world` and `www.zaratan.world`. Neither this period's top-pages list nor the query list shows a www-hostname duplicate, and `pages/_app.jsx:41` sets a `rel="canonical"` tag site-wide. Worth confirming in GSC's URL Inspection tool that the www host has dropped out of the index, but the click/impression split is gone from the data.
- **"daniel kronovet"** — new to the top-queries list: 6 clicks / 52 impr / 11.54% CTR / pos 3.6. A branded personal-name search converting well, likely riding founder visibility (Substack, press mentions).
- **`/chorewheel/start`** — new to top pages: 2 clicks / 34 impr / 5.88% CTR / pos 5.4.

## What is underperforming

- **"zaratan" brand query** — 358 impressions (tied for the 2nd-highest of any query) but only 2 clicks, 0.56% CTR, pos 8.2. The homepage (`pages/index.jsx`) sets no `pageTitle` prop, so it falls back to the generic default (`"Zaratan Coliving"`, `pages/_app.jsx:22-24`) instead of a title tailored to the "zaratan" query.
- **"settle living nyc"** — 185 impr, 1 click, 0.54% CTR, pos 5.0. Flagged in the prior report at 198 impr / 0.51% CTR / pos 5.6 as "the clearest snippet/title problem" on the list. Unresolved — position even improved slightly, CTR didn't move.
- **`blog.zaratan.world/p/two-months-at-settleliving`** — the single biggest traffic driver, but declining: 38 clicks / 1636 impr / 2.32% CTR / pos 5.8, down from 42 clicks / 1439 impr / 2.92% / pos 6.2. Impressions up 14%, clicks down 10% — CTR is leaking even as reach grows and position holds.
- **`/houses/sage`** — 11 clicks / 423 impr / 2.60% CTR / pos 10.5, down from 13 clicks / 261 impr / 4.98% / pos 12.1. Impressions up 62%, position improved by 1.6 spots, but CTR nearly halved and clicks fell. Still page 2.
- **Chore Wheel / Sage-name query cluster, zero clicks for a second consecutive report.** "chore wheel" (21 impr, pos 9.0), "chore wheel meaning" (8 impr, pos 8.6, improved from 10.8), "sage house" (62 impr, pos 6.5), "sagehouse" (36 impr, pos 8.1) — all page-1-adjacent, none converting. Same issue flagged last report, still unaddressed.
- **Desktop position degraded**: pos 13.5, down from 10.5 last report (mobile held roughly flat at 6.6 vs 7.0). Desktop CTR is also the weakest segment at 1.16%.

## Recommendations

1. **Give the homepage a real title tag.** Target: `zaratan.world/` for query "zaratan" (358 impr, 0.56% CTR, pos 8.2). Change: add a `pageTitle` in `pages/index.jsx`'s `getStaticProps` (currently absent, so it falls back to the generic default in `pages/_app.jsx:22-24`) — something concrete like "Zaratan Coliving — Shared Housing & Chore Wheel, Los Angeles." Expected impact: brand searches are high-intent; even a modest CTR lift (0.56% → 3%) recovers ~9 clicks/mo for a one-line change. Effort: **S**.

2. **Fix the CTR gap on "settle living nyc."** Target: query "settle living nyc" (185 impr, pos 5.0, 0.54% CTR), served by `blog.zaratan.world/p/two-months-at-settleliving`. Change: work with whoever controls the Substack post to add "NYC" and a review/comparison hook into the title, since that's the ranking intent this query isn't matching. This has now sat unresolved across two reports. Expected impact: pos 5.0 with a typical page-1 CTR (5–8%) is 9–15 clicks/mo vs. the current 1. Effort: **S**, contingent on Substack title being editable.

3. **Ship the pending `/houses/sage` comparison-table row.** Target: `/houses/sage` (423 impr, pos 10.5, CTR down to 2.60%). Change: `spec/CO-GTM.md` already flags the "Lease commitment (mutual, 4mo + M2M)" comparison-table row and FAQ reframe as the page's "highest-leverage edit," not yet shipped (`pages/houses/sage.jsx:117-135`, FAQ at line 155). Fresh, more complete on-page content plus internal links from `/about` and the homepage should help close the page-2 gap; this is a product/copy fix already scoped, not new SEO work. Effort: **M**.

4. **Audit Chore Wheel + Sage-name page titles and internal links.** Target: `/chorewheel`, `/chorewheel/start`, and the queries "chore wheel," "chore wheel meaning," "sage house," "sagehouse" (~127 combined impressions this period, all page-1-adjacent, zero clicks). Change: align title/meta on `/chorewheel/start` and `/chorewheel` with these exact query phrasings, and add internal links from relevant blog posts (`the-story-of-chore-wheel`, `chores-as-complex-coordination`) to `/chorewheel/start`. Effort: **M**.
