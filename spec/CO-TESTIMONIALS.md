# Colving -- resident testimonials

Verbatim quotes from Sage residents (current or departing).
Primary voice source for the `write-listing` skill: drafts borrow this language instead of inventing warmth.
Keep quotes verbatim — don't polish them; the roughness is the value.
Prompt that works: "What would you tell a friend who was thinking about moving in?"

## Format

```
### <who — e.g. departing resident, 2 yrs> (<YYYY-MM>)
- Context: <how it came up — exit chat, Slack, asked directly>

> the quote, verbatim
```
